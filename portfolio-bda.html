Wikipedia Toxicity

Project information

  • Category: Big Data Analytics
  • Category: Academic Project
  • Project date: 30 January, 2023
  • Project URL: https://github.com/stutigarg3/wikipedia-toxicity

In this project, I focused on leveraging the Wikipedia Talk Labels dataset, which consists of approximately 160,000 comments from Wikipedia's talk pages, meticulously labeled as toxic or non-toxic. The primary objective is to employ this dataset for text classification tasks, specifically hate speech detection and toxic comment classification. My preliminary data exploration reveals that this dataset poses some unique challenges, notably a significant class imbalance, with non-toxic comments accounting for the majority. This project aims to address these challenges by using various machine learning techniques, including TensorFlow and Keras for deep learning-based models and Scikit-Learn for traditional algorithms, to develop accurate classifiers. We plan to employ hyperparameter tuning to optimize model performance, considering the dataset's unbalanced nature. The insights derived from this project can contribute to improving online conversation quality, identifying and moderating toxic comments, and combating online harassment.

For data acquisition and preparation, I have already collected the labeled comments, and our data cleaning efforts have ensured that the dataset is ready for analysis. We have visualized the distribution of comment lengths, which indicates a positively skewed pattern with comments of varying lengths. The main tools in our arsenal include TensorFlow, Keras, and Scikit-Learn, which allow us to build and evaluate various machine learning models. As we progress, we will develop classifiers that can accurately predict whether a comment is toxic or non-toxic, and our evaluation will encompass metrics like accuracy, precision, recall, F1-score, and AUC-ROC. This project provides an opportunity to compare the performance of deep learning and traditional machine learning algorithms, ultimately aiming to create a robust and generalizable solution for hate speech detection and toxic comment classification.
