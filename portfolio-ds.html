Hotel Cancellation Predictions

Project information

  • Category: Data Science and Machine Learning
  • Category: Academic Project
  • Project date: 30 January, 2022
  • Project URL: https://github.com/stutigarg3/hotel-cancellation

This project is a captivating exploration of hotel and resort cancellations, brought to life through the prowess of data analysis and predictive modeling. It all begins with a comprehensive dataset, which paints a vivid picture of a fictitious resort. Packed with customer demographics, resort amenities, booking details, and satisfaction ratings, this dataset is our canvas. We harnessed the power of R to work our magic, using it for data acquisition, cleaning, transformation, and predictive modeling.

The journey started with data preprocessing, employing R's versatile functions like read.csv() for data loading, and conducting meticulous data cleaning to ensure data consistency. In the data transformation phase, we crafted new features to enhance our predictive model's performance. Machine learning models were the stars of the show, developed using popular libraries like caret, randomForest, and svm. These models were fine-tuned through cross-validation and hyperparameter tuning to optimize their performance. It's a project that showcases the beauty of data and the tools that unlock its secrets, from data acquisition to predictive modeling.
